Fresno State Library Badge & Certificate Verification

This page helps you confirm that a badge or certificate was legitimately issued by the Fresno State Library to verify completion of library tutorials. Each certificate includes a unique verification link that confirms its authenticity
(for example: https://kekoziar.github.io/fresnoStateLibrary-Badges/badge.html?id=TEST12345678901).

What a verification page shows

  • JSON format badge verification
  • Who the badge or certificate was issued to ("recipientName")
  • What was earned ("narrative")
  • When it was issued ("issuedOn")
The verification page displays information directly from Fresno State Library's issuing program supporting library tutorials, and cannot be edited by the recipient.
For questions or comments, contact Kat Koziar at kkoziar@mail.fresnostate.edu. Further information is at Library Tutorials and certificates FAQ on Fresno State Library Guides
